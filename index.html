Making responsive Website

Media Queries are a great way to ensure that all users of a website have a good experence. The media queries - literally '@media' as if it's asking the media a question - will ask the browser about it's current dimensions. The browser will tell our CSS what size itis, and the CSS will rearrange and re-display based on those parameters.
